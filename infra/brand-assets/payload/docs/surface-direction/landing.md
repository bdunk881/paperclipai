# Landing Page Visual Direction — v0.1.0

## Core Aesthetic: "The Electric Lab"
The landing page should feel like the "Front Door" to a sophisticated technical laboratory. It needs to be high-trust, high-speed, and developer-forward.

## Visual Rules
- **Stack:** Next.js + Sanity.
- **Background:** `Obsidian Dark` (#0f172a) for hero sections. Alternating Slate-950 for content sections.
- **Hero Treatment:** 
  - Dynamic "Neon Trace" background elements.
  - Large, bold typography (`Inter` or `Geist`).
- **Accent Logic:** 
  - Heavily lean on `Electric Teal` (#14b8a6) to symbolize the "spark" of automation.

## UI Patterns
- **MCP Badges:** Standardized badges for all MCP-compliant toolkits.
- **Integration Tiles:** High-fidelity stylized logos of the most popular integrations.
- **Copy Alignment:** Technical humanism. Professional but not stuffy.
