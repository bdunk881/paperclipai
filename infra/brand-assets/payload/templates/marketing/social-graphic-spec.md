# Marketing Asset Template: Social Media Graphic (v0.1.0)

## Dimensions
- **Standard Post:** 1200px x 630px (LinkedIn/Twitter)
- **Square:** 1080px x 1080px (Instagram/Discord)

## Grid & Composition
- **Safe Zone:** 64px padding all around.
- **Logo Placement:** Always bottom-right or top-left. Never centered unless it's the hero element.
- **Typography Layout:** Left-aligned or Center-aligned. Prefer Left-aligned for technical content.

## Visual Elements
- **Background:** `Obsidian Dark` (#0f172a) with subtle noise texture.
- **Accents:** 
  - Top-right corner: `Electric Teal` gradient glow (20% opacity).
  - Bottom-left corner: `AutoFlow Indigo` gradient glow (20% opacity).
- **Nodes:** Use the workflow node style (280px cards, 12px radius) to illustrate automation concepts.

## Export Settings
- Format: PNG-24 (lossless).
- Quality: High.
