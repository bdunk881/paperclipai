# Documentation Visual Direction — v0.1.0

## Core Aesthetic: "The Field Guide"
Documentation should be minimalist, high-legibility, and strictly focused on information architecture. It should feel like a well-organized technical manual for a high-end piece of hardware.

## Visual Rules
- **Background:** Slate-950 or White (depending on toggle). Standardize on the "Laboratory White" (#f8fafc) for light mode.
- **Typography:** 
  - Body: `Inter` (optimal for long-form reading).
  - Code: `JetBrains Mono` with syntax highlighting matched to the "Electric Lab" palette.
- **Sidebars:** Clean, simple hierarchy. No unnecessary shadows.

## UI Patterns
- **Callouts:**
  - `Info`: `AutoFlow Indigo`.
  - `Warning`: `Cyber Orange`.
  - `Success`: `Electric Teal`.
- **Integrations:** Every integration page must feature the canonical logo from the brand repo.
